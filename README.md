# MediLog 診療日誌

個人醫療就診記錄追蹤 App，幫助你清楚管理每次看診的資訊。

## 功能

- 📝 **新增／編輯就診記錄** — 記錄醫院、科別、醫師、診斷、用藥等完整資訊
- 📌 **狀態追蹤** — 待回診、需追蹤、已完成、持續治療中
- 📄 **PDF 匯出** — 一鍵產生就診報告 PDF
- 📤 **分享功能** — 複製格式化文字，方便分享給家人或其他醫師
- 💾 **本地儲存** — 資料存於瀏覽器，無需帳號

## 技術架構

- React 18 + Vite
- CSS Modules
- jsPDF（PDF 產生）
- LocalStorage（資料持久化）

## 快速開始

```bash
npm install
npm run dev
```

## 部署

連接此 repo 到 [Vercel](https://vercel.com) 即可自動部署。
